# MusicFlow
一个Android音乐播放器，可获取网易云音乐 & QQ音乐在线资源

解析在线资源的方法在mepride.club有详述

目前还有如下没做好：

- 通知栏
- 滚动歌词
- 离线下载
- 搜索页
- 私人FM
